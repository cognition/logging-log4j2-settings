## Azure Monitor / Log Analytics Integration

This directory contains example assets for ingesting logs from this project into Azure Monitor / Log Analytics.

- `dcr-hdfs-logs.json` — example Data Collection Rule (DCR) for host and container logs.
- `deploy-dcr.sh` — example script to deploy the DCR.
- `queries/` — KQL queries for various scenarios.

These assets are intended as starting points; adapt them to your environment.
